# Micro Services
Micros services for mobile app route-track-mobi

## Install and Run

```bash

npm install

npm run start
```

## Services End Points

1. Login 

  ```bash
  /signin
  ```

2. SignUp 

   ```bash
  /signup
  ```

3. Delete Account   
  
  ```bash
  /deleteaccount
  ```

5. Create Tracks 
  ```bash
  /tracks 
  ```

5. Delete Tracks 

  ```bash
  /tracks/delete
  ```
