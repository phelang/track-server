# Micro Services
Micros services for mobile app route-track-mobi

## Install and Run

```bash

npm install

npm run start
```

## Services End Points

1. Login 
  
  /signin

2. SignUp 
  
  /signup

3. Delete Account   

  /deleteaccount

5. Create Tracks 

  /tracks

5. Delete Tracks 

  /tracks/delete
